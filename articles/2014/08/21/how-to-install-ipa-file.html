如何安装ipa文件

1.打开iTunes软件，然后将iPhone连接到电脑上。连接成功之后iTunes界面右上方出现iPhone按钮，点击这个按钮跳转到手机信息的界面。

2.点击右下方的同步按钮，同步手机和电脑的应用。

3.将获得到的ipa文件放置在桌面上，双击打开。

4.切换到应用程序标签，在应用程序列表中搜索星期五。

5.点击安装按钮，按钮会变成将要安装。

6.再次点击同步按钮，同步完成之后，应用会出现在手机上。
